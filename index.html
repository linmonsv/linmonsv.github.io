How to build libiconv with arm compiler

Posted on 2021-06-17 Edited on 2022-07-04 In operating-system , linux , Linux , ARM

Download source code

https://www.gnu.org/software/libiconv/#downloading"

Build libiconv

tar zxvf libiconv-1.16.tar.gz
cd libiconv-1.16/
./configure –host=arm-linux-gnueabihf
make
find ./ -name ‘*.so’
./lib/.libs/libiconv.so
